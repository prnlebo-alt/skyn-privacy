Skyn Privacy Policy

Last updated: May 6, 2026

Skyn is a Chrome extension that applies visual CSS skins to supported AI assistant websites. Skyn is designed for appearance customization only.

What Skyn Does

Skyn injects local CSS styles into supported websites so users can customize the visual appearance of their chat interface. Supported websites may include Claude, ChatGPT, and Gemini.

What Skyn Does Not Do

Skyn does not:

  • Read, collect, store, or transmit your conversations
  • Access message text, prompts, responses, or chat history
  • Access input fields, textareas, or contenteditable elements
  • Listen for keystrokes or keyboard events
  • Access cookies, localStorage, or sessionStorage on supported websites
  • Inject ads or tracking scripts
  • Sell or transfer user data to third parties

Data Collection

Skyn does not collect personal information, health information, financial information, authentication information, personal communications, location data, web history, user activity, or website content.

Extension Storage

Skyn uses Chrome extension storage to save local extension preferences, such as the selected skin, selected font, enabled state, unlocked skins, license state, and optional background image settings. These settings are used only to provide the extension's customization features.

License and Purchase Verification

If paid features are used, Skyn may contact the Skyn backend to verify license status or purchase status. These requests are used only to determine which skins or features are unlocked. Payment processing is handled by Stripe. Skyn does not process or store credit card numbers.

Remote Code

Skyn does not execute remote code. All extension JavaScript is bundled inside the extension package. Network requests are used only for license and purchase verification.

Third-Party Websites

Skyn works on supported third-party websites by applying CSS for visual customization. Skyn is not affiliated with Anthropic, OpenAI, Google, Claude, ChatGPT, or Gemini.

Data Sharing

Skyn does not sell, rent, or transfer user data to third parties. Skyn does not use user data for advertising, creditworthiness, lending, or unrelated purposes.

Children's Privacy

Skyn is not directed to children under 13, and Skyn does not knowingly collect personal information from children.

Changes to This Policy

We may update this Privacy Policy from time to time. Any changes will be posted on this page with an updated date.

Contact

For privacy or support questions, contact: support@skynapp.app
